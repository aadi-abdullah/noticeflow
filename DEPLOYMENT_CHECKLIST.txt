7. **Wait 2-3 minutes** for build to complete

## ⚠️ COMMON GOTCHAS

### ❌ GitHub repo is PRIVATE
→ Change to PUBLIC (Streamlit Cloud requires it)
→ Go to repo settings → Change visibility

### ❌ Secrets not loading
→ Check format: `KEY = "value"` (with space around =)
→ Wait 30 seconds after saving
→ Refresh browser

### ❌ ".streamlit/secrets.toml" visible in GitHub
→ SECURITY ISSUE — rotate all API keys immediately
→ Go to .gitignore, add: `.streamlit/secrets.toml`
→ Run: `git rm --cached .streamlit/secrets.toml`
→ Commit: `git commit -m "Remove secrets from history"`

### ❌ "ModuleNotFoundError" on Streamlit
→ Missing package in requirements.txt
→ Run locally: `pip install [package]`
→ Update: `pip freeze > requirements.txt`
→ Git push (auto-redeploy)

### ❌ "API key invalid"
→ Check key is correct in Streamlit Secrets
→ Wait 1 minute and refresh
→ Check token hasn't expired
